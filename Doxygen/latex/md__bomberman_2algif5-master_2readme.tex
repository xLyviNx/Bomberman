\doxysubsection*{Installation}

Unfortunately I\textquotesingle{}m bad with build systems and makefiles and so on, so instead just do this\+: 
\begin{DoxyItemize}
\item Place \doxylink{algif_8h_source}{algif.\+h} somewhere where your compiler will find it. 
\item When compiling your code, make sure \doxylink{algif_8c_source}{algif.\+c}, \doxylink{bitmap_8c_source}{bitmap.\+c}, \doxylink{gif_8c_source}{gif.\+c} and \doxylink{lzw_8c_source}{lzw.\+c} are compiled along. 
\end{DoxyItemize}I also included a makefile in the .zip to create the example program but it probably only works in Linux. Allegro 5 (including the primitives addon) is required. \doxysubsection*{Examples}

The download comes with an example program.
